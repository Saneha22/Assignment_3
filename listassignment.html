Tags:

HTML uses a system of tags to defines the structure and layout of web pages.
tags are the building blocks of HTML documents and consists of angle brackets enclosing specific keywords.

  1. Specialised tags

    1. Head

      This is the head section of HTML document which contains meta-information about the document

    2. Title

      The title tag is like a label for a webpage.It appears at the top of your web browser or on the tab of the page

    3. Body

      This is the body section of the HTML document,which contains the visible content of the web page,such as heading ,paragraphs and more

      1. H1

        2. This is the heading tag used to define a top level heading.

      2. p

        3. This is a paragraph tag used to define a paragraph of text.

      3. hr

        This is a thematic break or seperator

      4. br

        5. It is a line break within the text

      5. span

        6. It is a inline container for text formatting.

      6. div

        7. it creates division or section in HTML

  2. Table tags

    1. Table

      2. The content of every table is enclosed by this tag

    2. td

      3. The smallest container in the table is a table cell,which we can create by using this tag.

    3. tr

      4. it states the row of table

    4. th

      5. it illustrates the table heading
