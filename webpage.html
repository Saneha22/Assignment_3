Features of HTML 5

HTML 5 is the newest version of HTML. It adds a diverse set of new tools for the web developer to use.
It incorporates all features from erlier versions of HTML. Some of them are:

  1. Introduction to Audio and Video: The audio and video tag are the two major addition to HTML 5. it allows users to embed a video and audio on their websites.
    HTML 5 video can use styling like CSS. You can change border, reflections, transitions, opacity etc. HTML 5 can make video super-fast and without having to build a video player.

  2. Header and footer: with the use of new tags header and footer , there is no lpnger a need to identify the two elements with "div" tag.
    Footer is placed at the end of the web page while header is placed at the start of the web page.
    By using these two tags, browser will know what to load first and what to load later.

  3. Figure and Figcaption: HTML 5 allows to use a figure element to mark up a photo in a document and a figcaption element to define a caption for the photo. This tag provides a container for content that is equivalent to a figure. It can be used to group a caption with one or more images.
