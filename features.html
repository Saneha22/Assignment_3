Features of HTML:5

  1. It is the latest version of HTML

  2. New Doctype: All HTML documents have a Doctype declaration.  Using the doctype declaration browesers,understood what document type to expext

  3. New Semantic Tags: New Semantic tags i.e, header and footer
    Because of this,the semantic tags need for "div" tag is eliminated for seperating two parts.

  4. Video and Audio tags: Developers can insert videos and photos by using these tags

  5. Forms: It enhanced the forms with new input types

  6. HTML:5 has new tags which commonly not found in previous one
    For example:
    "figure" tag is used to mark up photos in a document.
    "figcaption" defines a caption for the images
  7. Developers can also add graphics and animations with"canvas" and "svg"

  8. Web storage: With the help of this HTML:5 feature, web applications can store their data locally within the browser on the client side

  9. There is better error handling in HTML:5

  10. It helps to improve multimedia capabilities
