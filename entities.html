HTML Entities

HTML Entities are the special codes that are used to display reserved characters, special characters, unicode characters.

  • It is used to display symbols that have special meanings in HTML
  • It also ensures the characters are rendered correctly

Some commonly used HTML entities

  1. for &, there are another special entity i.e, "amp ;"
  2. for <, there are another special entity i.e, "It"
  3. for >, there are another special entity i.e, "gt"
  4. for ", there are another special entity i.e, "quot"
  5. for ', there are another special entity i.e, "apos;"
